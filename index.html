Installation Windows

  1. Télécharge le fichier .zip
  2. Clic droit → Extraire tout…
  3. Ouvre le dossier extrait
  4. Double-clique sur le .exe
⚠️ Ne lance pas le .exe depuis le zip.
Si Windows bloque : “Informations complémentaires” → “Exécuter quand même”.
